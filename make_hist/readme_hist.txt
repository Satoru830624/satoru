-------------------------------------------------
                射影画像生成ツール
-------------------------------------------------

1. はじめに
このツールは、指定されたフォルダ内の画像に対してX方向およびY方向の射影を取得し、その結果を別のフォルダに保存します。

2. 必要なライブラリのインストール
このツールを使用する前に、PythonのOpenCVとNumPyライブラリが必要です。
以下のコマンドでインストールしてください:

pip install opencv-python numpy

3. 使用方法

- 入力画像が保存されているフォルダのパスと、結果を保存するフォルダのパスをプログラム内で指定します。
input_dir = "入力画像が保存されているフォルダのパス"
output_dir = "結果を保存するフォルダのパス"

- 指定したフォルダに画像を配置し、プログラムを実行します。
python3 hist.py

- 実行すると、入力フォルダ内の画像すべてに対して射影を計算し、結果を出力フォルダに保存します。

4. 出力結果
出力フォルダには、各入力画像に対して2つの射影画像が保存されます:
- {画像名}_x_projection.png: X方向の射影画像
- {画像名}_y_projection.png: Y方向の射影画像

5. 注意事項
- このツールは、.png、.jpg、.jpeg形式の画像のみをサポートしています。
- 出力結果のフォルダは、プログラムの実行前に存在している必要はありません。存在しない場合は自動的に作成されます。
